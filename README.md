# ✈️ Flight Delay Prediction using Machine Learning

A machine learning project that predicts whether a flight is likely to be delayed based on historical flight data. The project applies data preprocessing, exploratory data analysis (EDA), feature engineering, and multiple classification algorithms to identify the most effective prediction model.

---

## 📌 Project Overview

Flight delays are influenced by several factors such as weather conditions, airport traffic, airline operations, and previous flight delays. This project analyzes a large-scale flight dataset and compares multiple machine learning models to predict flight delays accurately.

---

## 🎯 Objectives

- Analyze historical flight data
- Perform Exploratory Data Analysis (EDA)
- Build and compare multiple machine learning models
- Identify the most important factors affecting flight delays
- Evaluate model performance using different metrics

---

## 📊 Machine Learning Models

- Logistic Regression
- Decision Tree

- Gradient Boosting

---

## 📈 Model Performance

| Model | Accuracy |
|--------|----------|
| Logistic Regression | **76.31%** |
| Decision Tree | **70.43%** |
| Random Forest | **80.28%** |
| Gradient Boosting | **81.36%** ⭐ |

Gradient Boosting achieved the highest prediction accuracy and was selected as the best-performing model.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Flight-Delay-Prediction.git
```

2. Navigate to the project folder

```bash
cd Flight-Delay-Prediction
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python src/flight_delay_prediction.py
```
